## Компонент dropdown

Vue-обёртка над Bootstrap dropdown с поддержкой поиска, прокрутки и режима select для замены ⟨select⟩. Детальная информация о компоненте (API, особенности реализации, примеры использования) находится в шапках файлов:
- Компонент: `shared/components/dropdown.js`
- Шаблон: `shared/templates/dropdown-template.js`

## Компонент combobox

Vue-обёртка над Bootstrap `input-group` + `dropdown` с поддержкой автодополнения, фильтрации, клавиатурной навигации, произвольного ввода и валидации. Детальная информация о компоненте (API, особенности реализации, примеры использования) находится в шапках файлов:
- Компонент: `shared/components/combobox.js`
- Шаблон: `shared/templates/combobox-template.js`

## Компонент dropdown-menu-item

Универсальный компонент пункта выпадающего меню с поддержкой иконки, текста, подзаголовка и суффикса. Детальная информация о компоненте (API, особенности реализации, примеры использования) находится в шапках файлов:
- Компонент: `shared/components/dropdown-menu-item.js`
- Шаблон: `shared/templates/dropdown-menu-item-template.js`

## Компонент button

Универсальный компонент кнопки с поддержкой иконки, текста и суффикса. Детальная информация о компоненте (API, особенности реализации, примеры использования) находится в шапках файлов:
- Компонент: `shared/components/button.js`
- Шаблон: `shared/templates/button-template.js`

## Компонент button-group

Vue-обёртка над Bootstrap `.btn-group` с поддержкой трёх типов кнопок: `button`, `checkbox` и `radio`. Детальная информация о компоненте (API, особенности реализации, примеры использования) находится в шапках файлов:
- Компонент: `shared/components/button-group.js`
- Шаблон: `shared/templates/button-group-template.js`
